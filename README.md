### [PHP/MySQL (オンライン書店システム - hung-bookstore)](./backend/hung-bookstore)

**概要:** PHPとMySQLを用いた書籍ECサイトです。ユーザーの購入から管理者の在庫・受注管理まで一連の流れを網羅しています。

**アピールポイント:**
- **フルスタック開発:** カート機能や在庫連動など、ECサイトの主要ロジックを実装。
- **管理機能の充実:** 割引率設定が可能なセール管理や注文ステータス更新機能を搭載。
